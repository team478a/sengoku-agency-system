# 共通ID検索・紐づけ確認画面 v3.6.65

## 目的

代理店システムをハブとして、ショッピングカート、戦国パスポート、ウォレット、今後追加される外部サービスのユーザーを共通IDで追跡できるようにします。

## 追加画面

- `/admin/common_id_mappings.php`
- 管理メニュー: `共通ID検索`

## 確認できる内容

- 共通ID一覧
- 外部サービス別ユーザーIDとの紐づけ
- 共通IDと代理店・プロジェクトの紹介関係
- 共通ID単位の外部連携ログ

## 主な検索キー

- `common_user_id`
- 外部サービスキー
- 外部サービス側ユーザーID
- 代理店コード、代理店名、担当者名
- ウォレットアドレス
- 状態

## 運用上の使い方

外部サービス側から登録されたユーザーが、どの代理店の紹介として扱われているかを確認できます。

例:

1. 戦国パスポートやカート側のユーザーIDで検索
2. 紐づいた `common_user_id` を確認
3. `紹介関係` 表示で代理店、プロジェクト、登録元を確認
4. 問題がある場合は `詳細ログ` でAPI連携履歴を確認

## 注意

このバージョンでは閲覧・検索に限定しています。
共通IDの手動マージ、紐づけ解除、代理店変更は誤操作の影響が大きいため、別フェーズで監査ログ付きの管理機能として実装する想定です。
